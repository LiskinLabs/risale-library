# Память проекта и Архитектурные решения (Risale-i Nur Digital Library v2)

## 🎯 Главная цель

Создание ультимативного инструмента для изучения трудов «Рисале-и Нур» на базе движка **Readest**. Приложение объединяет в себе профессиональную читалку EPUB (уровня Яндекс Книг) и исследовательскую среду для глубокого погружения в исламские тексты.

---

## 🏗️ Архитектурный фундамент (Движок Readest)

_Мы перешли на базу Readest (Next.js + Tauri v2 + Rust + SQLite), что дает нам:_

- **Кроссплатформенность:** Единый код для Windows, macOS, Android и iOS.
- **Ядро чтения:** Профессиональный рендеринг через `foliate-js`.
- **Производительность:** Полнотекстовый поиск на Rust (Tantivy).
- **Офлайн-первичность:** Локальная база данных SQLite.

---

## 🗺️ Дорожная карта и уникальные фичи

### 1. Интеллектуальный слой Рисале-и Нур

- **Параллельное чтение (Split-screen):** Одновременное отображение оригинала (турецкий/османский) и русского перевода.
- **Интерактивный Люгат 2.0:** Глубокая интеграция словаря терминов. При клике на слово — не просто перевод, а контекстное толкование из "Рисале".
- **Связь с Первоисточниками:** Кросс-ссылки на Аяты и Хадисы с открытием карточки оригинала (арабский + тафсир).
- **Морфологический поиск:** Поиск по корням слов (находит "Хакикат" при поиске по корню Х-К-К).

### 2. Система обучения и удержания

- **Флэш-карточки (Spaced Repetition):** Возможность добавлять незнакомые термины в систему повторения (аналог Anki) прямо из текста.
- **Audio-Sync (Whispersync):** Синхронизация текста с аудиокнигами. Подсветка читаемого фрагмента в реальном времени.
- **Трекинг Вирдов:** Встроенные планы чтения ("Прочитать 'Слова' за месяц") с визуализацией прогресса.

### 3. Социальный и Экспортный функционал

- **Quote Cards:** Генератор красивых карточек с цитатами для соцсетей (Telegram/Instagram).
- **Экспорт исследователя:** Выгрузка хайлайтов и заметок в форматах Markdown, Word или PDF.
