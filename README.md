[![Platform](https://img.shields.io/badge/platform-macOS%20%7C%20Windows%20%7C%20Linux%20%7C%20iOS%20%7C%20Android%20%7C%20Web-lightgrey)]()

Risale Digital Library — это open-source платформа для чтения и изучения исламских текстов, созданная специально для работы с коллекцией Рисале-и Нур. Приложение объединяет мощный движок чтения электронных книг с инструментами глубокого анализа текста и AI-ассистентом.

Платформа работает на **macOS, Windows, Linux, Android, iOS и Web** с синхронизацией прогресса, заметок и библиотеки между устройствами.

## Возможности

### 📖 Чтение и навигация
| Возможность | Описание |
|---|---|
| **Мультиформатный движок** | Поддержка EPUB, PDF, MOBI, TXT, CBZ с настраиваемыми шрифтами, темами и режимами отображения |
| **Параллельное чтение** | Изучение оригинального текста и перевода/комментариев бок о бок |
| **Режим скорочтения (RSVP)** | Быстрое чтение с настраиваемой скоростью и отслеживанием прогресса |
| **Линейка чтения** | Подсветка строк для фокусировки внимания |
| **Непрерывная прокрутка** | Прокрутка без разрывов страниц с сохранением позиции |
| **Навигация по сноскам** | Всплывающие сноски с историей переходов |

### 🤖 AI Ассистент
| Возможность | Описание |
|---|---|
| **Google Gemini интеграция** | Мгновенный перевод текста, объяснение контекста и интеллектуальный поиск |
| **Контекстный анализ** | AI-помощник понимает контекст изучаемого текста |
| **Умный перевод** | Перевод выделенных фрагментов с сохранением терминологии |

### ✍️ Аннотации и заметки
| Возможность | Описание |
|---|---|
| **Продвинутое выделение** | Множество стилей выделения с настраиваемыми цветами и именами |
| **Закладки и заметки** | Мгновенное добавление заметок к выделенным фрагментам |
| **Глобальные выделения** | Создание аннотаций, действующих на все вхождения фрагмента в книге |
| **Экспорт** | Экспорт аннотаций в plain text, Readwise и другие форматы |
| **Лупа при выделении** | Увеличительное стекло для точного выделения текста |

### 📚 Управление библиотекой
| Возможность | Описание |
|---|---|
| **Встроенная библиотека Рисале-и Нур** | Полная коллекция текстов Рисале-и Нур с правильным форматированием |
| **Группировка** | Организация книг по авторам, сериям и категориям |
| **Импорт** | Пакетный импорт книг с фильтрацией по формату и размеру |
| **OPDS интеграция** | Подключение к OPDS-каталогам с авто-загрузкой новых книг |
| **Внешние папки** | Чтение книг из внешних папок без копирования в библиотеку |

### 📖 Встроенный словарь (Lügat)
| Возможность | Описание |
|---|---|
| **Мгновенный поиск** | Поиск определений сложных терминов и османской лексики |
| **MDict и StarDict** | Импорт пользовательских словарей в форматах MDict и StarDict |
| **Системные словари** | Использование встроенных словарей macOS, iOS и Android |
| **Wikipedia интеграция** | Быстрый доступ к Wikipedia для дополнительного контекста |

### 🔄 Синхронизация
| Возможность | Описание |
|---|---|
| **Кроссплатформенная синхронизация** | Синхронизация прогресса чтения, заметок и закладок через Supabase + Turso |
| **KOReader синхронизация** | Полная интеграция с KOReader: синхронизация библиотеки, закладок и аннотаций |
| **Hardcover.app** | Интеграция с Hardcover для отслеживания прогресса чтения |

### 🎧 Доступность и удобство
| Возможность | Описание |
|---|---|
| **Text-to-Speech (TTS)** | Озвучивание текста с подсветкой читаемого предложения |
| **Тёмная тема** | Полноценная тёмная тема с настройкой цветов |
| **E-ink режим** | Оптимизация для электронных чернил (e-ink устройств) |
| **Настраиваемые шрифты** | Поддержка пользовательских шрифтов с синхронизацией между устройствами |
| **Режим фокусировки** | Настраиваемые линейки чтения и минималистичный интерфейс |

### 🔒 Безопасность
| Возможность | Описание |
|---|---|
| **PIN-блокировка** | Опциональная 4-значная блокировка при запуске |
| **Изолированный рендеринг** | Контент книг рендерится в sandboxed iframe |
| **Шифрование передачи** | Весь синхронизационный трафик использует HTTPS/TLS |

Скачайте последнюю версию для вашей платформы со страницы [Releases]():

| Платформа | Формат |
|---|---|

### Системные требования

- **macOS:** 12.0 (Monterey) или новее
- **Windows:** 10/11 (x64 или ARM64)
- **Linux:** Современный дистрибутив с WebKitGTK
- **Android:** 8.0+ (SDK 26+)
- **iOS:** 15.0+

## Разработка

### Предварительные требования

- **Node.js** (v24+) и **pnpm** (v11+)
- **Rust** и **Cargo** (для Tauri сборок)
- **Supabase** проект (для авторизации)
- **Turso** база данных (для синхронизации)
- **Google Gemini API Key** (для AI функций)

```bash
# Установка Node.js и pnpm
nvm install v24
nvm use v24
npm install -g pnpm

# Установка Rust
rustup update
```

### Быстрый старт

```bash
# 1. Клонирование репозитория
git clone https://github.com/LiskinLabs/risale-library.git
cd risale-library
git submodule update --init --recursive

# 2. Установка зависимостей
pnpm install
pnpm --filter @LiskinLabs/app setup-vendors

# 3. Настройка окружения
# Создайте .env и .env.local в apps/readest-app/ на основе .env.example

# 4. Запуск для разработки
pnpm dev-web          # Web версия
pnpm tauri dev        # Десктопная версия (Tauri)

# 5. Мобильная разработка
pnpm tauri android dev  # Android
pnpm tauri ios dev      # iOS (требуется macOS + Xcode)
```

### Структура проекта

```
risale-library/
├── apps/
│   ├── readest-app/          # Основное приложение (Next.js + Tauri)
│   │   ├── src/              # Исходный код фронтенда
│   │   ├── src-tauri/        # Tauri (Rust) бэкенд
│   │   └── extensions/       # Браузерное расширение
│   └── readest.koplugin/     # KOReader плагин (Lua)
├── packages/
│   ├── foliate-js/           # Движок рендеринга EPUB/PDF
│   ├── tauri/                # Форк Tauri фреймворка
│   ├── tauri-plugins/        # Плагины Tauri
│   └── js-mdict/            # Парсер словарей MDict
├── docker/                   # Docker/Podman конфигурация
├── data/                     # Метаданные, скриншоты, иконки
└── .github/                  # CI/CD workflows
```

### Доступные команды

| Команда | Описание |
|---|---|
| `pnpm dev-web` | Запуск Web версии для разработки |
| `pnpm tauri dev` | Запуск Tauri десктопной версии |
| `pnpm build-web` | Сборка Web версии |
| `pnpm tauri build` | Сборка десктопной версии |
| `pnpm test` | Запуск тестов (Vitest) |
| `pnpm lint` | Проверка кода (Biome + TypeScript) |
| `pnpm format` | Форматирование кода |

### Docker (Self-Hosting)

```bash
cd docker
cp .env.example .env
# Отредактируйте .env с вашими настройками
docker compose up -d
```

Приложение будет доступно на `http://localhost:3000`.
